### Exercise 1: React Model
- **Objective**: Understand the React component model and the virtual DOM.
- **Content**: Theoretical explanation of how React works under the hood.

### Exercise 2: Environment Setup
- **Objective**: Learn how to set up a React development environment.
- **Content**: Step-by-step guide to setting up a React project using Vite.

### Exercise 3: Functional Components and JSX
- **Objective**: Create and use functional components with JSX.
- **Content**: Examples of user profiles displayed using functional components.

### Exercise 4: Props
- **Objective**: Understand data flow in React using props.
- **Content**: Demonstrates passing data to components and validating props with PropTypes.

### Exercise 5: State
- **Objective**: Manage component memory using state.
- **Content**: Includes a counter example and a conceptual question about state vs. props.

### Exercise 6: User Interaction
- **Objective**: Handle user input and events in React.
- **Content**: A login form with controlled inputs and form submission handling.

### Exercise 7: Composition
- **Objective**: Design advanced components using composition.
- **Content**: Accordion component demonstrating state lifting and controlled panels.

### Exercise 8: Debugging with React DevTools
- **Objective**: Use React Developer Tools for debugging.
- **Content**: Instructions for installing and using React DevTools to inspect components and state.

### Capstone: To-Do List Application
- **Objective**: Build a fully functional To-Do List app.
- **Content**: Features include adding, toggling, and deleting tasks, with state management and props hierarchy.
